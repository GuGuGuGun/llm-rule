# AGENTS.md

给 AI 编码 Agent 的工作规则。

少改代码 · 改对问题 · 先验证 · 不乱加复杂度 · 默认中文

---

`AGENTS.md` 让 AI Agent 在代码库里更像一个可靠的资深协作者：先理解目标，再做最小正确变更，最后用测试和 diff 证明结果。

## 它做什么

* 约束 Agent 默认用中文、直接、简洁地回复
* 要求代码修改优先采用最小正确变更
* 避免重复实现、隐藏回退和第二套事实来源
* 鼓励非平凡改动使用 TDD 和自动化验证
* 要求最终检查 diff，说明改了什么、验证了什么
* 禁止执行会影响远程仓库的 Git 操作

## 核心理念

好的 Agent 不靠“改了很多代码”证明价值。

它应该靠：

* 清楚理解问题；
* 克制地修改代码；
* 删除或复用优先于新增；
* 用测试验证行为；
* 交付容易审查的 diff。

最好的补丁通常是小的、直接的、无惊喜的。
